1. Apa Itu Perangkat Lunak Komputer?

"Software is the general term for various kinds of programs used to operate and manipulate computers and their peripheral devices."
(Introduction to Information System by James A. O'Brien and George M. Marakas).

Dalam buku "Introduction to Information System" karya James A. O'Brien dan George M. Marakas (edisi ke-15) dijelaskan bahwa perangkat lunak adalah istilah umum untuk berbagai jenis program yang digunakan untuk mengoperasikan dan memanipulasi komputer dan perangkat periferalnya.

Dikutip dari idcloudhost.com , perangkat lunak atau software adalah data yang diprogram, disimpan, dan diformat secara digital dengan fungsi tertentu. Perangkat ini sendiri tidak memiliki bentuk fisik, hanya bisa dioperasikan lewat perangkat komputer. Untuk pembuatannya sendiri, perangkat ini memerlukan bahasa pemrograman yang ditulis oleh seorang programmer yang ahli akan bidang tersebut. Selanjutnya, perangkat tersebut dikompilasikan memakai aplikasi kompiler agat dapat menjadi kode yang bisa dikenali oleh mesin hardware atau perangkat keras. Yang dimaksud hardware disini adalah Personal Computer.

Secara umum, fungsi perangkat lunak adalah sebagai berikut:

  • sebagai dasar kebutuhan komputer agar dapat dioperasikan dengan baik.
  • mengatur hardware/perangkat keras yang ada pada komputer. Dengan begitu, komputer yang digunakan dapat bekerja dengan baik sesuai pekerjaan apa yang dibutuhkan.
  • sebagai penghubung antara beberapa software yang lain dengan hardware komputer.
  • sebagai penerjemah perintah pada software lain yang ada dalam bahasa mesin. Dengan begitu, hardware pada komputer pun bisa mengerti dan menerimanya dengan baik.
  • digunakan dalam mengindentifikasi sebuah program di dalam komputer.

Sederhananya, perangkat lunak komputer adalah sebuah program komputer yang digunakan untuk melakukan fungsi fungsi tertentu, seperti menjadi sistem operasi agar pengguna bisa menjalankan komputer dengan baik, sebagai program untuk menjalankan fungsi tertentu pada hardware, dan masih banyak kegunaan perangkat lunak yang lainnya.

2. Jenis-jenis Perangkat Lunak Komputer

James A. O'Brien dan George G. Marakas dalam bukunya yang berjudul "Introduction to Information System" membagi perangkat lunak menjadi 2 jenis, yaitu Application Software dan System Software, dan dari kedua jenis tersebut terbagi lagi menjadi jenis-jenis software yang lebih spesifik lagi.

A. Application Software

"application software is software that directs the performance of a particular use, or application, of computers to meet the information processing needs of users. Application software includes a variety of programs that can be segregated into general purpose and application-specific categories."
(Introduction to Information System by James A. O'Brien and George M. Marakas).

perangkat lunak aplikasi adalah perangkat lunak yang mengarahkan kinerja penggunaan atau aplikasi tertentu dari komputer untuk memenuhi kebutuhan pemrosesan informasi pengguna. Perangkat lunak aplikasi mencakup berbagai program yang dapat dipisahkan ke dalam kategori tujuan umum (General Purpose Application Programs) dan aplikasi khusus (Application-Specific Programs).

a. General Purpose Application Programs

"Programs that can perform information processing jobs for users from all application areas."
(Introduction to Information System by James A. O'Brien and George M. Marakas).

Program yang dapat melakukan pekerjaan pemrosesan informasi untuk pengguna dari semua area aplikasi. Maksudnya, jenis software ini bisa dan biasa digunakan oleh semua orang atau pihak yang menggunakan komputer, tidak ada batasan tertentu tentang siapa saja yang menggunakan jenis software ini, selama pengguna bisa menggunakan komputer dengan baik, pengguna tersebut harusnya bisa untuk memanfaatkan software jenis ini. Ada beberapa kategori untuk jenis software ini, diantaranya :

  • Software Suites
    " A combination of individual software packages that share a common graphical user interface and are designed for easy transfer of data between applications."

    Kumpulan dari paket software individu yang berbagi/memiliki interface yang mirip dan dirancang untuk memudahkan transfer data antar aplikasi. Software-software ini biasanya memiliki fungsi yang saling berkaitan satu sama lain. Contohnya adalah Microsoft Office, Microsoft Office merupakan Kumpulan software yang yang memiliki interface dan cara penggunaan yang mirip.

  • Web Browsers
    "A software package that provides the user for accessing Internet, intranet, and extranet Web sites. Browsers are becoming multifunction universal clients for sending and receiving e-mail, downloading files, accessing Java applets, participating in discussion groups, developing Web pages, and other Internet, intranet, and extranet applications."

    Sebuah paket software yang memungkinkan pengguna untuk mengakses situs Web Internet, intranet, dan ekstranet. Browser menjadi klien universal multifungsi untuk mengirim dan menerima email, mengunduh file, mengakses applet Java, berpartisipasi dalam kelompok diskusi, mengembangkan halaman Web, dan aplikasi Internet, intranet, dan ekstranet lainnya. Beberapa contoh software web browser ini adalah Google Chrome, Mozila Firefox, Microsoft Edge, dan Opera.

  • Electronic Mail
    "Sending and receiving text messages between networked PCs over telecommunications networks. E-mail can also include data files, software, and multimedia messages and documents as attachments."

    Mengirim dan menerima pesan teks antara PC melalui jaringan telekomunikasi. E-mail juga dapat menyertakan file data, perangkat lunak, dan pesan multimedia dan dokumen sebagai lampiran. Contoh software untuk software layanan email ini adalah Gmail dan Microsoft Outlook.

  • Word Processing
    "The automation of the transformation of ideas and information into a readable form of communication. It involves the use of computers to manipulate text data to produce office communications in the form of documents."

    Otomatisasi transformasi ide dan informasi menjadi bentuk komunikasi yang mudah dibaca. Ini melibatkan penggunaan komputer untuk memanipulasi data teks untuk menghasilkan komunikasi kantor dalam bentuk dokumen. Contoh word processing sofftware adalah Microsoft Word, LibreOffice Writter dan WPS Word.

  • Spreadsheets
    "An application program used as a computerized tool for analysis, planning, and modeling that allows users to enter and manipulate data into an electronic worksheet of rows and columns."

    Sebuah program aplikasi yang digunakan sebagai alat komputerisasi untuk analisis, perencanaan, dan pemodelan yang memungkinkan pengguna untuk memasukkan dan memanipulasi data ke dalam lembar kerja elektronik baris dan kolom. Contoh software Spreadsheets adalah Microsoft Excel, LibreOffice Calc, dan WPS Spreadsheets.

  • Database Managers
    "Database managers is software designed to facilitate data manipulation and database structure management. It also acts as a gate between the database, the users and apps that want to view or change the data."

    Manajer basis data adalah perangkat lunak yang dirancang untuk memfasilitasi manipulasi data dan manajemen struktur basis data. Ini juga bertindak sebagai gerbang antara database, pengguna dan aplikasi yang ingin melihat atau mengubah data. Contoh software database managers adalah Microsoft Access dan MySQL.

  • Presentation Graphics
    Presentation Graphics is a software using computer-generated graphics to enhance the information presented in reports and other types of presentations.

    Presentasi Grafik adalah perangkat lunak yang menggunakan grafik yang dihasilkan komputer untuk meningkatkan informasi yang disajikan dalam laporan dan jenis presentasi lainnya. Contoh software Presentation graphics adalah Microsoft PowerPoint, LibreOffice Impress, dan WPS Presentation.

  • Personal Information Managers
    A software package that helps end users store, organize, and retrieve text and numerical data in the form of notes, lists, memos, and a variety of other forms.

    Paket perangkat lunak yang membantu pengguna menyimpan, mengatur, dan mengambil data teks dan numerik dalam bentuk catatan, daftar, memo, dan berbagai bentuk lainnya. Contoh softwarenya adalah Akeneo dan Salsify.

  • Groupware
    Software to support and enhance the communication, coordination, and collaboration among networked teams and workgroups, including software tools for electronic communications, electronic conferencing, and cooperative work management.

    Software untuk mendukung dan meningkatkan komunikasi, koordinasi, dan kolaborasi di antara tim jaringan dan kelompok kerja, termasuk software untuk komunikasi elektronik, konferensi elektronik, dan manajemen kerja kooperatif. Contoh softwarenya adalah Google Workspace, Microsoft Outlook dan Slack.

b. Application Specific Programs

Application software packages that support specific applications of end users in business, science and engineering, and other areas.
(Introduction to Information System by James A. O'Brien and George M. Marakas).

Paket perangkat lunak aplikasi yang mendukung secara spesifik pengguna dalam bisnis, sains dan teknik, dan area lainnya. Singkatnya, jenis software aplikasi ini digunakan oleh pihak tertentu saja, berguna untuk mendukung pengguna di bidang tertentu secara spesifik. Ada beberapa kategori untuk jenis software ini, diantaranya :

  • Business Accounting, Transaction Processing, Customer Relationship Management, Enterprise Resource Planning, Electronic Commerce

    Kumpulan software untuk membantu keberlangsungan dan kelancaran bisnis yang sedang dilakukan. Contohnya adalah software-software dari Zoho, seperti Zoho CRM, Zoho Projects dan Zoho Creator.

  • Science and Engineering

    Software yang digunakan di bidang science dan engineering contohnya adalah SciNote yang biasa digunakan sebagai Electronic Lab Notebook.

  • Education

    Software standar yang bisa digunakan secara khusus di bidang pendidikan contohnya adalah Google Classroom sebagai kelas virtual.

B. System Software

Programs that control and support operations of a computer system.
(Introduction to Information System by James A. O'Brien and George M. Marakas).

Program yang mengontrol dan mendukung operasi sistem komputer. System software merupakan jenis software yang dirancang untuk berkomunikasi dengan hardware atau bahkan application software. Ini bertindak sebagai interface antara hardware dan program komputer di mana ia mengkoordinasikan tugas-tugas antara dua komponen sistem komputer. Perangkat lunak sistem mencakup berbagai program yang dapat dipisahkan ke dalam kategori Program Manajemen Sistem (System Management Programs) dan Program Pengembangan sistem (system Development Programs).

a. System Management Programs

"Programs that manage the hardware, software, network, and data resources of computer systems during the execution of the various information processing jobs of users. Examples of important system management programs are operating systems, network management programs, database management systems, and system utilities."
(Introduction to Information System by James A. O'Brien and George M. Marakas).

Program yang mengelola perangkat keras, perangkat lunak, jaringan, dan sumber daya data sistem komputer selama pelaksanaan berbagai pekerjaan pemrosesan informasi pengguna. Contoh program manajemen sistem yang penting adalah sistem operasi, program manajemen jaringan, sistem manajemen basis data, dan utilitas sistem."

  • Operating Systems
    "The main control program of a computer system. It is a system of programs that controls the execution of computer programs and may provide scheduling, debugging, input/output control, system accounting, compilation, storage assignment, data management, and related services."

    Program utama dari sistem komputer. Ini adalah sistem program yang mengontrol pelaksanaan program komputer dan dapat menyediakan penjadwalan, debugging, kontrol input/output, akuntansi sistem, kompilasi, tugas penyimpanan, manajemen data, dan layanan terkait. Contoh Operating System yang populer digunakan adalah Microsoft Windows, Machintos OS, dan berbagai distro dari Linux.

  • Network Management Programs
    "Network management programs perform functions such as automatically checking client PCs for input/output activity, assigning priorities to data communications requests from clients and terminals, and detecting and correcting transmission errors and other network problems."

    Program manajemen jaringan melakukan fungsi seperti secara otomatis memeriksa PC klien untuk aktivitas input/output, menetapkan prioritas untuk permintaan komunikasi data dari klien dan terminal, dan mendeteksi dan memperbaiki kesalahan transmisi dan masalah jaringan lainnya. Contoh softwarenya adalah SolarWinds Network Configuration Manager (NCM).

  • Database Management Systems
    • Database management systems are software packages that simplify the creation, use, and maintenance of databases. They provide software tools so that end users, programmers, and database administrators can create and modify databases; integrate a database; generate reports; do application development; and perform database maintenance.

    Sistem manajemen basis data adalah paket perangkat lunak yang menyederhanakan pembuatan, penggunaan, dan pemeliharaan database. Mereka menyediakan perangkat lunak sehingga pengguna, programmer, dan administrator database dapat membuat dan memodifikasi basis data; mengintegrasi database; menghasilkan laporan; melakukan pengembangan aplikasi; dan melakukan pemeliharaan database. Contoh softwarenya adalah Microsoft SQL Server.

  • Application Servers
    application server, software that provides a middleware interface between the operating system and user application programs.

    server aplikasi, perangkat lunak yang menyediakan antarmuka middleware antara sistem operasi dan program aplikasi pengguna. Server Aplikasi menyediakan layanan dan infrastruktur untuk mengembangkan, menyebarkan, dan menjalankan aplikasi. Contoh softwarenya adalah Microsoft IIS Oracle WebLogic dan NGINX.

  • System Utilities
    • System utilities software are programs that perform routine computing functions, such as backing up data or copying files, as part of an operating system or as a separate package

    Perangkat lunak utilitas sistem adalah program yang melakukan fungsi komputasi rutin, seperti mencadangkan data atau menyalin file, sebagai bagian dari sistem operasi atau sebagai paket terpisah. Contohnya adalah CCleaner dan Restoro.

b. System Development Programs

Programs that help users develop information system programs and procedures and prepare user programs for computer processing. Major software development programs are programming language translator, Programming editor, and a ariety of CASE (computer-aided software engineering) and other programming tools.
(Introduction to Information System by James A. O'Brien and George M. Marakas).

Program yang membantu pengguna mengembangkan program dan prosedur sistem informasi dan menyiapkan program pengguna untuk pemrosesan komputer. Program pengembangan perangkat lunak utama adalah penerjemah bahasa pemrograman, editor Pemrograman, dan berbagai CASE (rekayasa perangkat lunak berbantuan komputer) dan alat pemrograman lainnya.

  • Programming Language Translators
    "programming language translators are programs that translate other programs into machine language instruction codes that computers can execute"

    penerjemah bahasa pemrograman adalah program yang menerjemahkan program lain ke dalam kode instruksi bahasa mesin yang dapat dieksekusi oleh komputer. Penerjemah bahasa pemrograman juga dapat merujuk ke kompiler, assembler, atau interpreter; atau apa pun yang mengubah kode tingkat yang lebih tinggi menjadi kode tingkat tinggi lainnya. Contoh softwarenya adalah MinGW, software tambahan sebagai compiler untuk bahasa C, C++, ADA, Fortran dan Java. Ada juga software DevC++ sebagai editor sekaligus compiler bahasa C++ agar program dalam bahasa C++ bisa dijalankan oleh mesin.

  • Programming Editors.
    "programming language editors, software that help programmers write programs by providing a variety of programs creation and editing capabilities."

    editor bahasa pemrograman, perangkat lunak yang membantu programmer menulis program dengan menyediakan berbagai kemampuan pembuatan dan pengeditan program. Secara garis besar, kategori ini terbagi menjadi 2 jenis, yaitu IDE dan Code Editor.

    IDE (Integrated Development Environment) adalah sosftware yang menyediakan interface untuk pengembangan kode, fitur pengujian dan debugging. Ini membantu mengatur artefak proyek yang relevan dengan kode sumber aplikasisoftware. Software ini menyediakan beberapa alat dan fitur untuk membuat pengembangan mudah dan standar berdasarkan bahasa pemrograman pengembang menulis kode. IDE juga memiliki fungsi untuk mengkompilasi dan menginterpretasikan program. Untuk IDE ini biasanya satu software hanya dikhususkan untuk satu teknologi saja. Contoh dari IDE adalah Microsoft Visual Studio, IntelliJ IDEA, dan Android Studio.

    Code Editor pada dasarnya adalah text editor tetapi juga dirancang untuk membantu Anda menulis kode. Ini membantu Anda mempercantik kode Anda dan memberi Anda alat yang lebih canggih untuk membuat pengkodean lebih mudah bagi Anda. Editor kode sederhana dapat berupa program perangkat lunak yang berdiri sendiri atau bagian dari IDE atau browser web. Contoh dari code editor adalah Microsoft Visual Studio Code, Sublime Text, dan Notepad++.

  • Computer-Aided Software Engineering(CASE) Packages
    "Computer-Aided Software Engineering (CASE), software packages to accomplish and automate many of the activities of software development or programming."

    Computer-Aided Software Engineering (CASE), paket perangkat lunak untuk menyelesaikan dan mengotomatiskan banyak aktivitas pengembangan atau pemrograman perangkat lunak.
    Computer-aided software engineering (CASE) adalah implementasi alat dan metode yang difasilitasi komputer dalam pengembangan perangkat lunak. CASE digunakan untuk memastikan perangkat lunak berkualitas tinggi dan bebas cacat. CASE memastikan pendekatan yang terarah dan disiplin serta membantu desainer, pengembang, penguji, manajer, dan lainnya untuk melihat pencapaian proyek selama pengembangan. Untuk contoh softwarenya, tersedia sangat banyak, namun terbagi-bagi sesuai kategori dan kebutuhan.
